## 元概念 · 借势合力

> 创业不是从零造势，而是识别已有的"势"，借势而行，合力而起。

**借势 = 看懂经济周期 × 顺势而为**
- 第五次康波（信息技术）萧条期 + 第六次康波（AI）萌芽期
- 两条最优路径：旧周期做极致性价比 / 新周期做创新十倍好
- 在两个周期的**交叉地带**最容易借力

**合力 = 回归原点 × 找到不变的内核**
- "染黑店"经典案例：和服市场崩了，回到能力原点（把布染到极致黑）→ 旧衣翻新
- 原力不是当前业务，是**底层能力**——载体可换，能力不变

**复利公式**：`势 × 力 = 复利`

## Gate 1 · 原力觉醒（Force Awakening）· 色 g1 红 `#C0392B`

**问的问题**：我是谁——找到不变的能量原点。

**定义**：向内探寻不变的内核，找到即使周期更替仍能稳坐钓鱼台的"能量原点"。

**关键 artifacts**：
- **三圈交集 2.0**：非理性痴迷 × 非对称优势 × 非线性回报
- **天才区定位**：做这件事时持续能量增益（不累反充电）
- **12 原型人格定位**（Jungian archetypes）
- **5 维非对称优势判词**：动能回收 / 努力产出 / 重复时长 / 认知偏差 / 模仿壁垒
- **6 种非理性痴迷信号**：时间失真 / 无悔回报 / 情绪牵引 / 反复回归 / 身份绑定 / 解释传道
- **生物结构三层驱动**：骑象人（理性决策）× 大象（感性动力）× 路径（隐知识/审美回路）
- **阴影 → 超能力翻转**：童年阴影/痛点反向长出当下能力
- **创始人母题（founding motif）**：一个动词（如"看见"/"翻译"/"守护"），载体可换、母题不变
- **A/B 框架**：同一个底层动力的两面——A 健康路径（建设性、可控）× B 破坏性反射（自动启动、消耗）

**触发条件 / 何时活跃**：
- 个人觉醒期、寻找内在 thesis
- 必须先于品类独创
- 防止"灵魂干涸"（创业三种死法之首）

**与其他周期关系**：
- 前置：必须先做完 → 才能进 Gate 2 品类独创
- 输出物：母题 → 给 Gate 2 当品类定位的原料
- 完成信号：能用"一句话天赋定位"讲清楚自己 + 通过 8 大访谈问题验证

## Gate 2 · 品类独创（Category of One）· 色 g2 蓝 `#2E5984`

**问的问题**：我为谁——锁定甜用户，解贵问题，开辟新品类。

**定义**：通过创造全新品类使产品无法被归类比价，从而占据用户心智的独占定价权。

**关键 artifacts**：
- **12 字真言**：无法分类 × 重新定义 × 自带预算 × 品类独创
- **脑科学 6 模块验证**：注意力捕获 → 情绪标签 → 因果叙事 → 多巴胺预测 → 框架效应 → 身份一致性
- **心理账户跃迁**：用户预算从"成本"账户迁到"情感/身份/投资"账户
- **甜用户画像**：年龄/职业/家庭层级/客单/决策人格（J/P 型、PM 型、艺术家型）
- **客户母题金字塔**：表层（功能）→ 场景层 → 深层（关系定位）→ 终层（心智需求）→ 底层（共通母题）
- **价值阶梯**：功能 → 情绪 → 社交 → 意义。意义层无溢价上限
- **一句入脑 killer one-liner**：能被原话复述、能传给朋友
- **第四种物种**：与既有 3 种同行画清楚差异化（X 是什么 + Y 是什么 + Z 是什么 + 你 = 全新组合）
- **品类验证四象限**：客户端付钱 ✅ / 搜索端有量 / 同行端开始抄 / 媒体端用你的词

**触发条件**：
- Gate 1 觉醒已完成，母题已清晰
- 市场扫描发现未满足的 JTBD（job to be done）
- 非对称优势对应到行业盲点

**与其他周期关系**：
- 上游：Gate 1 的母题作输入
- 下游：Gate 3 用品类的定价权 → 跑出可持续 unit economics
- 完成信号：陌生人付费 + 真实复购意图 + 用户开始用你的新词

## Gate 3 · 模式升维（Business Model Elevation）· 色 g3 绿 `#4A7C59`

**问的问题**：怎么赚——前后财三链路 × 五层资源结构。

**定义**：对商业模式进行三链路系统诊断与重构，实现从"人肉交付"到"自动赚钱系统"的跃迁。

**关键 artifacts**：
- **前链路（获客 / Desirability）**：客户为什么/通过什么渠道找到你
- **后链路（交付 / Feasibility）**：能否标准化交付，是否依赖创始人
- **财链路（利润 / Viability）**：unit economics、毛利、规模化下能否维持
- **39 题精简诊断 + 12 题快速用户访谈**
- **60 天测试**："如果创始人离开 60 天，业务会不会塌？"
- **五层资源结构**：主投线（真投资源）/ 半投线（小资源测试）/ 机会主义线（不主动开发）/ 远景线（0 资源占位）/ 生产工具升级（跨线杠杆）
- **三层防白嫖咨询**：免费决策建议 / 收费方向方案 / 签约后完整方案
- **场景互导飞轮**：轻交付做漏斗 + 重交付赚利润 + 零交付攒资产
- **贝叶斯验证 C/B/A/S**：想法 C → 朋友说好 B → 陌生人付费 A → 付费且转介绍 S
- **收入金字塔**：S 级标杆 / A+/A 利润引擎 / 品牌空间第二曲线 / 轻产品漏斗 / 内容资产复利

**触发条件**：
- Gate 2 品类已被早期客户付费验证
- 创始人触顶："我自己 scale 不出去"
- 需要找到可重复、可委派的核心运营

**与其他周期关系**：
- 上游：Gate 2 的品类定价权 + JTBD 作输入
- 下游：Gate 4 把三链路的优势 → 沉淀成 16 控制点
- 完成信号：60 天创始人离开业务不塌 + unit economics 正 + 10× 需求下毛利仍 sustain

## Gate 4 · 壁垒锁定（Barrier Lock-In）· 色 g4 紫 `#6B5B95`

**问的问题**：怎么守——虚实入出 × 16+1 控制点。

**定义**：通过虚实入出四维 16 个战略控制点构建穿越周期的资产护城河。

**关键 artifacts**：
- **30 法则**："一家伟大公司，未来十年赚的钱比过去十年更多。"
- **四维度**：
  - **虚壁垒 V**（品牌文化 · 供应端软实力）：V1 品牌溢价 / V2 特许经营 / V3 品类独占 / V4 文化叙事
  - **实壁垒 R**（规模效率 · 供应端硬实力）：R1 规模效应 / R2 链主强控 / R3 工艺黑箱 / R4 极致成本（V8 新增 R5 服务可信度）
  - **入壁垒 I**（转换成本 · 需求端粘性）：I1 学习成本 / I2 数据沉淀 / I3 沉没成本 / I4 生态依赖
  - **出壁垒 O**（网络效应 · 需求端扩张）：O1 病毒系数 / O2 双边市场 / O3 网络价值 / O4 数据飞轮
- **总记分卡**：四维各打 1-10 分 + 雷达图 + 综合分（目标 5.5/10）
- **主攻 / 第二主攻 / 维护 / 补短板** 四档资源分配（不要 16 个全主攻）
- **AI 时代新壁垒**：数据飞轮——用得越多越聪明，是康波六的新增维度

**触发条件**：
- 模式跑通 + 单位经济可规模化
- 竞争对手开始冒头
- 从"不惜代价增长"切换到"建立可防御优势"

**与其他周期关系**：
- 上游：Gate 3 的三链路优势作输入
- 终态：进入"系统假设期"，可衍生原力人生（Force-as-Life-OS）
- 完成信号：竞品 18 个月内无法复制 + 70%+ 毛利 sustain + NPS > 60 + 品牌成为品类指代词

## 元结构 · 4 周期作为螺旋（不是线性）

```
Cycle 1 · 原型假设期（Prototype）  ——→ 原力觉醒
Cycle 2 · 价值假设期（Value）      ——→ 品类独创
Cycle 3 · 增长假设期（Growth）     ——→ 模式升维
Cycle 4 · 系统假设期（System）     ——→ 壁垒锁定 + 原力人生
```

每周期内含 4 个验证节点 → 4 × 4 = 16 步螺旋。**不能跳级**，跳级会触发"创业三种死法"：
- 灵魂干涸（跳过 Gate 1）
- 融资陷阱（跳过 Gate 3）
- 三年放弃（跳过 Gate 4）
